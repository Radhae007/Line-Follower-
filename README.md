# Line-Follower-
A  Arduino-based line follower robot utilizing an 8-sensor QTR array. This project implements PID (Proportional-Integral-Derivative) control for smooth line tracking and dedicated logic for handling 90-degree turns and intersections.

#Components
Microcontroller- Arduino Uno/Nano/Esp32 /ESP 8266
Sensors -QTR-8A (Analog) Reflectance Sensor Array
Motor Driver- TB6612FNG Dual Channel Driver
Motors - 2x N20 Gear Motors 
Chassis -Custom Printed
